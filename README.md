# E-Commerce Frontend

Frontend del proyecto de PP1 con React, HTML, CSS y JavaScript.
Ofrece una interfaz moderna, dinámica y responsiva que permite a los usuarios navegar por productos, ver detalles, gestionar el carrito de compras y realizar pedidos de forma sencilla.

---

## Tecnologías utilizadas

- Frontend: React con Next.js y CSS para los formularios y validaciones.

- Backend: Node.js con Express.

- Base de datos: MySQL (con ORM o consultas SQL directas).

- Control de versiones: GitHub para trabajo colaborativo.

---

## Requisitos

- Node.js (v18 o superior)
- npm o yarn

---

```bash
# Clonar el repositorio
git clone https://github.com/usuario/nombre-del-repositorio.git

# Acceder al directorio del proyecto
cd nombre-del-repositorio
